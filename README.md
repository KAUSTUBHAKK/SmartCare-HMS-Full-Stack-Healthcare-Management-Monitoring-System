# Medicare.AI

Full-stack AI-assisted healthcare management platform.

## Structure

```text
MedicareAI-GitHub/
├── frontend/      React frontend
├── backend/       Node.js Express backend
├── ai-service/    Python FastAPI service
├── README.md
└── .gitignore
```

## Tech Stack

- React, JavaScript, HTML, CSS
- Node.js, Express.js, MongoDB Atlas, Mongoose, JWT
- Python, FastAPI, Uvicorn
- TensorFlow.js, Tesseract.js, Socket.IO, jsPDF

## Main Features

- Patient, doctor, and admin login
- Appointment booking
- AI health chat
- Symptom checker
- Image disease screening
- Medicine OCR
- AI report PDF
- Medical timeline
- Health records
- Medicine reminders
- English, Hindi, Marathi language support
